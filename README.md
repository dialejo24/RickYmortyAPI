## Configuración y Ejecución

Sigue los siguientes pasos para poner en marcha el proyecto en entorno local:

### 1. Clonar el proyecto
```bash
git clone <url-del-repositorio>
cd rickYmorty
```

### 2. Instalar dependencias
Es necesario descargar todas las librerías necesarias antes de ejecutar la aplicación:
```bash
npm install
```

### 3. Iniciar el servidor de desarrollo
Para ejecutar la aplicación en modo de desarrollo:
```bash
npm run dev
```
Una vez ejecutado, la terminal mostrará una dirección (normalmente `http://localhost:5173`). Ábrela en el navegador para ver la aplicación.
